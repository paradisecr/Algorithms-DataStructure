# Algorithms and Data structure practice.

1. package说明：  

**src/main**下为类实现目录。

**src/test**为对应测试目录。

使用Maven作为项目管理工具，使用Junit包进行测试工作，每个实现类在test目录下对应的包中有相应的测试用例。
